Projectname: Covid tracker for India and it's states

This project has been built Using ReactJS. The Date for the project has been fetched from an online serving API. The data has been represented in the form of different charts. Initially when the User lands on the Page the total data for India Including Confirmed, Recovered , And Deaths will be displayed to the User. The User then have got the previlage to Select Different States from the State Dropdown. The Values for the Dropdown are also fetched from the API. As soon as the User selects the state the Respective districts of the state selected will be displayed to the User. Once the User selects the District values from the District Dropdown Confirmed Number of Cases will be displayed. Whenever Users Cahnges the State value from the dropdown the Charts will also get refreshed for those corresponding State Values.

Projectname: React-Quiz-App

This is a Complete Fullstack project (MERN) and Typescript. The Data for the application has been fetched from an online Quiz API. In Order to get access to the Quiz Section Users first need to register themselves and once successful signup users will recieve a confirmation email. Users also have the previlage to reset their password if they have forgotten the password and then can proceed with the login. Routes of the Application are secured i.e if the user doesn't login and want to see the Home Page he will be straightaway redirected to Signin Page and if the User is Logged in and want to go to Signup page then only option is to logout and if user tries to access the Signup /Signin route he will be straightaway redirected to Home Page. Backend of the Application is designed using Node and Express and Frontend is done with React (Typescript and Redux ).
